# Data Science & Machine Learning Learning Journey

## Overview
Welcome to my Data Science and Machine Learning learning repository! I am using this workspace to track my progress through a comprehensive 25-day curriculum. This folder contains all my practice materials, Jupyter notebooks, datasets, and personal notes as I work my way from Python programming basics to building machine learning models and participating in Kaggle competitions.

## Learning Progress Tracker
The curriculum is divided into a 25-day intensive syllabus. I will use the checkboxes below to track my progress:

### Part 1: Python Programming & Data Structures
- [ ] **Day 1: Introduction to Python Programming** - Overview of Python, installing Python/Anaconda, Jupyter Notebook, variables, data types, and basic I/O.
- [ ] **Day 2: Python Data Structures & Control Flow** - Lists, Tuples, Dictionaries, Sets, conditional statements (`if`, `else`), loops (`for`, `while`), `lambda`, `map()`, `filter()`, and list comprehensions.

### Part 2: Data Manipulation & Analysis
- [ ] **Day 3: NumPy Essentials** - Creating arrays, reshaping, indexing & slicing, mathematical operations, broadcasting, and aggregation.
- [ ] **Day 4: Pandas Series & DataFrames** - Loading data (CSV, Excel), and exploring data (`head`, `info`, `describe`).
- [ ] **Day 5: Data Cleaning with Pandas** - Handling missing data (`fillna`, `dropna`), dropping columns, and replacing values.
- [ ] **Day 6: Advanced Pandas** - Merging, grouping (`groupby`), aggregations (`sum`, `mean`, `count`), pivot tables, multi-index, joining, and concatenating.

### Part 3: Data Visualization
- [ ] **Day 7: Data Visualization Basics** - Introduction to Matplotlib, line charts, bar charts, and scatter plots. Choosing the right chart.
- [ ] **Day 8: Advanced Visualization** - Customizing plots (titles, legends, grids) and introduction to Seaborn (histograms, KDE, boxplots).
- [ ] **Day 9: Relationships & Correlation** - Pairplots, heatmaps, and correlation matrices.
- [ ] **Day 10: Time Series Analysis** - Resampling and rolling windows.

### Part 4: Statistics & SQL
- [ ] **Day 11: Statistics Fundamentals** - Measures of central tendency (mean, median, mode), dispersion (variance, standard deviation), and probability distributions concepts.
- [ ] **Day 12: SQL Databases (Part 1)** - Fundamentals of SQL, `SELECT`, `WHERE`, `ORDER BY`, and `GROUP BY`.
- [ ] **Day 13: SQL Databases (Part 2)** - SQL Joins (`INNER`, `LEFT`, `RIGHT`) and Subqueries.

### Part 5: Applied Exploratory Data Analysis (EDA)
- [ ] **Day 14: EDA Project** - Hands-on exploratory data analysis project (e.g., Titanic or Sales dataset). Perform EDA and extract actionable insights.

### Part 6: Machine Learning (Supervised & Unsupervised)
- [ ] **Day 15: Introduction to Machine Learning** - What is ML? Supervised vs. Unsupervised learning, ML workflow, and train/test splits.
- [ ] **Day 16: Linear Regression** - Theory, implementation with Scikit-Learn, and evaluation metrics (RMSE, R²).
- [ ] **Day 17: Logistic Regression & Classification** - Classification problems, implementation, and evaluation metrics (Accuracy, Precision, Recall, F1-score).
- [ ] **Day 18: Naive Bayes & Support Vector Machines (SVM)** - Gaussian & Multinomial Naive Bayes, SVM margin/kernels.
- [ ] **Day 19: Decision Trees** - Theory, implementation, and tree techniques.
- [ ] **Day 20: k-Nearest Neighbors (k-NN)** - Concept, distance metrics, and finding the optimal 'k'.
- [ ] **Day 21: Clustering & Dimensionality Reduction** - K-Means clustering and Principal Component Analysis (PCA).

### Part 7: Model Optimization & Ensembles
- [ ] **Day 22: Hyperparameter Tuning** - `GridSearchCV`, `RandomSearchCV`, and Cross-Validation.
- [ ] **Day 23: Ensemble Methods** - Random Forest and Gradient Boosting.

### Part 8: Review & Capstone
- [ ] **Day 24: Course Review & Interview Prep** - Review of all algorithms, common ML interview questions, and introduction to Kaggle.
- [ ] **Day 25: Final Capstone Project** - Build an end-to-end ML project (e.g., Housing Prices, Credit Risk) and submit to a leaderboard.
